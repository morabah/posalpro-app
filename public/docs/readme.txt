POSALPRO MVP2 - QUICK START GUIDE
==================================

Welcome to PosalPro MVP2!

This is a sample README file for testing document preview functionality.

SYSTEM REQUIREMENTS
-------------------
- Node.js 18 or higher
- PostgreSQL 15 or higher
- Redis 7 or higher
- 4GB RAM minimum
- 10GB disk space

INSTALLATION STEPS
------------------
1. Clone the repository
2. Run 'npm install'
3. Copy .env.example to .env.local
4. Configure database connection
5. Run 'npm run db:seed'
6. Start development server: 'npm run dev:smart'
